Extends the MVP bandits (\S5 of \texttt{mvp\_specification.tex}) by conditioning action selection on the current activity level.

\section{Formulation}

Context $c_t = s_t.\texttt{activity} \in \{\texttt{sedentary}, \texttt{active}\}$. Instead of one $(\alpha_a, \beta_a)$ per action, maintain $(\alpha_{c,a}, \beta_{c,a})$ for each context-action pair. Sample $\theta_{c_t, a} \sim \text{Beta}(\alpha_{c_t, a}, \beta_{c_t, a})$, pick $a^* = \arg\max_a \theta_{c_t, a}$. Update only the observed $(c_t, a_t)$ posterior.

\section{Optimal Policy}

The optimal contextual policy (from the transition matrices in \S3) is: nudge when sedentary ($0.3 > 0.1$), idle when active ($0.6 > 0.5$). Stationary distribution: $\pi_s = 4/7$, $\pi_a = 3/7$.

\[
E[r] = \tfrac{4}{7} \cdot 0.3 + \tfrac{3}{7} \cdot 0.6 = \tfrac{3}{7} \approx 0.429 \text{ per step} \implies 192.9 \text{ total}
\]

Best non-contextual (always nudge): $E[r] = 0.375$, total $168.8$. The contextual bound is $14.3\%$ higher.

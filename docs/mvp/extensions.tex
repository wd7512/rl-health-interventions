Extends the MVP bandits (\S5 of \texttt{mvp\_specification.tex}) by conditioning action selection on the current activity level.

\section{Formulation}
Context $c_t = s_t.\texttt{activity} \in \{\texttt{sedentary}, \texttt{active}\}$. Each algorithm maintains separate parameters per $(c, a)$ pair instead of global per-action parameters:

\begin{itemize}
  \item \textbf{Thompson Sampling:} $(\alpha_{c,a}, \beta_{c,a})$ posteriors. Sample $\theta_{c_t, a} \sim \text{Beta}(\alpha_{c_t, a}, \beta_{c_t, a})$, pick $a^* = \arg\max_a \theta_{c_t, a}$.
  \item \textbf{Epsilon-Greedy:} $Q_{c,a}$ values. With probability $\epsilon$ explore, otherwise pick $a^* = \arg\max_a Q_{c_t, a}$.
  \item \textbf{UCB:} $Q_{c,a}$ values with per-context counts. $a^* = \arg\max_a \left[ Q_{c_t, a} + c \sqrt{\ln(N) / n_{c_t, a}} \right]$ where $N$ is global step count.
\end{itemize}

Update only the observed $(c_t, a_t)$ pair.

\section{Optimal Policy}
The optimal contextual policy (from the transition matrices in \S3) is: nudge when sedentary ($0.3 > 0.1$), idle when active ($0.6 > 0.5$). Stationary distribution: $\pi_s = 4/7$, $\pi_a = 3/7$.

\[
E[r] = \tfrac{4}{7} \cdot 0.3 + \tfrac{3}{7} \cdot 0.6 = \tfrac{3}{7} \approx 0.429 \text{ per step} \implies 192.9 \text{ total}
\]

Best non-contextual (always nudge): $E[r] = 0.375$, total $168.8$. The contextual bound is $14.3\%$ higher.

# Plan Change

Use this skill to make a small, practical implementation plan.

## Workflow

1. State the goal and expected outcome.
2. Inspect the relevant files before proposing edits.
3. List the smallest set of files likely to change.
4. Choose the minimal correct implementation approach.
5. Define verification commands before editing.
6. After implementation, update the plan if evidence changes.

## Output

- Goal
- Files likely affected
- Implementation steps
- Verification steps
- Open questions, if any
